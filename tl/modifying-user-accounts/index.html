Opisyal na Pangtaong Manwal ng DRIVER

To access the main site, go to roadsafety.gov.ph

Baguhin ang mga User Account

May kakayahan ang mga Admin ng DRIVER na baguhin ang mga user account gaya ng pagpapalit ng kanilang username at e-mail. Maaari ring baguhin ng Admin ang access privilege ng isang user para maging Admin din o Analyst o Public Viewer.

 

Baguhin ang E-mail, Access Privilege, at Username ng isang User

Para baguhin ang Access ng isang user sa DRIVER, pindutin ang “Manage Users” sa kanang itaas ng Ashlar Editor. Pagkapindot, lalabas ang listahan ng lahat ng user ng DRIVER.

Mula rito, hanapin ang e-mail ng user. Magagawa ito sa pamamagitan ng pagpindot ng Ctrl + F sa Windows at Cmd + sa Mac para mapadali ang paghahanap.


Sa kanang bahagi ng bawat user, makikita ang “Edit.” Pindutin lamang ito para mapunta sa page na “Modify User Account” kung saan maaari nang palitan ang e-mail, username, at access privilege ng user. Pindutin ang “Save Changes” para ma-save ang mga ginawang pagbabago.

Pagtanggal ng Users

Mula ulit sa page na User Management, hanapin ang e-mail ng user na nais tanggalin sa DRIVER. Pagkahanap, pindutin lamang ang “Delete” sa kanan ng e-mail ng user.

 

Matatanggal na ang user sa DRIVER kapag pinindot ito.

Opisyal na Pangtaong Manwal ng DRIVER

To access the main site, go to roadsafety.gov.ph

Uri ng Related Content

Sa Ashlar Editor, “Related Content” ang tawag sa mga kategoryang makikita sa Input forms sa DRIVER.

Ang mga halimbawa ng Related Content ay “Vehicles, People, at Photos.” Sa loob ng bawat Related Content ay mga fields at datos na kailangang kolektahin at kalaunan ay gagamitin upang masagot ang “Incident Input Form.” Ang “Incident Input Form” ay ang ulat na ginagamit sa mga insidente o “Incidents.”

Mababago ng mga Admin ang Related Content gamit ang Ashlar Schema Editor.

Siyasatin ang mga Uri ng Related Content

Makikita ang mga uri ng Related Content sa pamamagitan ng pagpindot sa “View Related Content” sa sidebar sa kaliwang bahagi ng screen. Pagkapindot, makikita ng Admin User ang listahan ng lahat ng Related Content na kabilang sa uri ng Record na iyon (sa halimbawang ito, makikita ang lahat ng Related Content na kabilang sa “Incidents.” Ito ay ang “Vehicles,” “People,” at “Photos”).

Baguhin ang Related Content

Maaaring baguhin ang Related Content sa pamamagitan ng pagpindot ng “Edit” na matatagpuan sa kanang bahagi ng bawat Related Content. Ang pagbabagong maaaring gawin sa bawat Related Content ay ang pagdaragdag ng mga input fields at kategory na gagamitin sa pagkumpleto ng Incident Report Form at kakailanganin sa pangongolekta ng datos. Makikita ang mga ginawang pagbabago sa DRIVER website at ito ngayon ay maaari nang magamit ng mga Analyst.

Sa pagdaragdag ng mga kategorya at fields, makakapili ang Admin User ng uri ng field na gusto nilang makita sa DRIVER. Halimbawa, kung gusto ng mga Admin User na magkaroon ng field na listahan, piliin ang “Select List.” Kapag ang nais ay malayang pagsagot ng field (sa paraang patalata o pangungusap), piliin ang “Text Field.” Makikita ang halimbawa sa larawan sa ibaba:

Pagdagdag ng Related Content

Maaaring magdagdag ng Related Content ang mga Admin user sa pamamagitan ng pagpindot ng “Add Related Content” na matatagpuan sa sidebar sa kaliwa ng Ashlar Editor. Maaari ring pindutin ang “Add New Content” button sa kanang itaas ng page para sa Related Content.

Maaaring idagdag ang pang-isahan o pang-maramihang anyo ng record. Kunwari, kung gustong magdagdag ng bagong uri ng record na “Incident,” ito ang ilalagay sa “Single Title,” samantalang “Incidents” naman sa “Plural Title.” Mag-aadjust ang mga ito ayon sa pagkakagamit sa DRIVER: kung isa o marami ang magiging record.

Maraming Record sa isang Related Content

 

Kapag minarkahan ang “Allow Multiple?” sa kaliwang ibabang bahagi ng page, mapapahintulutan ang Analyst na makapagdagdag ng maraming record sa isang Related Content. Halimbawa, kung gusto ng Admin na makapagrecord ang Analyst ng maraming “Vehicle,” kailangang markahan ng Admin ang “Allow Multiple?”

Pagtanggal ng Related Content

Para matanggal ang isang Related Content kailangan lamang pindutin ang “Trash” sa kanang bahagi ng screen.

Driver /tl Online User Manual Fri, 08 Dec 2017 10:28:08 +0000 tl hourly 1 Maging Analyst /tl/become-an-analyst/ /tl/become-an-analyst/#respond Tue, 10 Oct 2017 10:55:29 +0000 /?p=1141 Read more →]]> Para makapag input, edit at mag-delete ng datos sa DRIVER website, dapat ang gumagamit ay mayroong pribilehiyong pang-Analyst. Ang mga sumusunod ang mga hakbang upang maging Analyst:

  1. Bago mag-register, magtalaga ng isang/grupo ng mga opisyal na maglalagay ng datos sa DRIVER.
  2. Gamitin ang Gmail account para mabuksan ang DRIVER. Paki sundan ang ayos sa ibaba:

roadsafety.<your city/municipality/province>@gmail.com

(Example: roadsafety.pasig@gmail.com)
